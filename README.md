# Ncmdump GUI
 A simple GUI for Ncmdump, a tool to decrypt NetEase Cloud Music's .ncm files.
 一个简单的 Ncmdump GUI，用于解密网易云音乐的 .ncm 文件。

## Features 特性
- Drag and drop to decrypt .ncm files 拖放解密 .ncm 文件
- Save cover image 保存封面图片
